Every action evaluated by hiitl produces an immutable audit record, including blocked actions. The audit API provides read access to these records.

## List audit records

```
GET /v1/audit
```

**Required scope:** `read`

Returns audit records matching the specified filters, ordered by timestamp descending.

**Query parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `agent_id` | string | Filter by agent ID |
| `action` | string | Filter by action/tool name |
| `decision` | string | Filter by decision type (`ALLOW`, `BLOCK`, etc.) |
| `environment` | string | Filter by environment |
| `start_date` | ISO 8601 | Records after this timestamp |
| `end_date` | ISO 8601 | Records before this timestamp |
| `user_id` | string | Filter by end-user ID |
| `limit` | integer | Max records to return (default 50, max 1000) |
| `cursor` | string | Pagination cursor from previous response |

**Example:**

```bash
curl -H "Authorization: Bearer $API_KEY" \
  "https://api.hiitl.com/v1/audit?agent_id=payment-agent&decision=BLOCK&limit=10"
```

### Pagination

Use cursor-based pagination for large result sets:

```bash
curl -H "Authorization: Bearer $API_KEY" \
  "https://api.hiitl.com/v1/audit?cursor=eyJsYXN0X2lkIjoiYXVkX2FiYzEyMyJ9&limit=50"
```

## Get audit record

```
GET /v1/audit/{id}
```

**Required scope:** `read`

Returns a single audit record with full detail including the complete envelope and decision response.

**Example:**

```bash
curl -H "Authorization: Bearer $API_KEY" \
  https://api.hiitl.com/v1/audit/aud_abc123
```

## Record integrity

Every audit record includes a SHA-256 `content_hash` computed at write time. This enables individual record tamper detection.

<Note>
Audit records are **append-only**. Once written, a record cannot be modified or deleted for its retention window. The `content_hash` provides cryptographic proof that a record has not been altered.
</Note>

## Retention

Audit records are retained according to org-level configuration:

| Tier | Default retention |
|------|-------------------|
| Free | 7 days |
| Starter | 30 days |
| Enterprise | Configurable (up to indefinite) |

Records past their retention window are automatically purged.
